Institution

SRI SAI RAM INSTITUTE OF TECHNOLOGY (Autonomous institution) is approved by AICTE,GoI.To add to its pride, the institution has been accredited with “A+” grade by NAAC. The institution has also reached 192th position amongst the Top 200 Engineering Institutions all over India by MHRD, GoI. In addition, the college has ranked for 3rd in SWACHH CAMPUS RANKING 2018, all over India.

Department

The Department of CSE is accredited by NBA, AICTE. Every year Department produces at least 10 rank holders and has Centre of Excellences for Red Hat, Cisco, Oracle, Salesforce, VMWare & Data Science and Big Data Analytics. The Department has signed MoU's with ICTACT, Red Hat Academy, Cisco Academy, Oracle Academy, SANDS India Pvt. Ltd., EPR Labs, IBM Rational Seed, Network Bulls Pvt.Ltd., and CSN Systems. Our students are active members of CSI,ISTE, SIESRP and IEI.

About

World's First & Virtual NUTPAM 2020!! Bounding the theme 'ANYONE CAN CODE', Sri Sairam Institute of Technology is hosting a one-day TechFest virtually to bring out talent's among budding engineers. It becomes interesting when code meets fun..!!

Events

TechWar

REGISTER

Description

  • The participants will be given a question with 3/4 th of the code (python preferably).
  • The participant has to read and understand the code and then solve it to get the workable solution.
  • Individual participation is only allowed.

Technovation

REGISTER

Description

  • The participants will be given problem statements from StudyMonk.
  • The participants must solve the problem within the given time.
  • Minimum 2 and maximum 3 persons are allowed

Battle of Design

REGISTER

Description

  • Exclusive event for designers and meme creators.
  • The participants will be given a page with a name of any app eg. Home page, login page.
  • The participant has to design a template with their own design in any photo editing software.
  • Individual participation

Grab The Infinites

REGISTER

Description

  • This technical event consists of 6 different programs and where each of the programs are interlinked to each other.
  • Initially the coder starts with a problem and if solves a problem then he will get a string output along with the next program he/she has to solve.
  • The clue or the solutions from the initial program is given as a test/custom input by the user to next program to get the next clue and next program he has to solve.So, this process continues until he solves the final program (6th one) and got the very last result revealed to the moderators.
  • The fastest hand wins the game.
  • All the coders must have "HackerRank "account and all the languages in HackerRank are allowed.
  • Individual participation.

Scholar's Tale

Conne"T"ech

REGISTER

Description

  • Connection game is a type of abstract strategy game in which the set of pictures will be projected to the players.
